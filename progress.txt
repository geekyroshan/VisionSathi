## Codebase Patterns

_Add reusable patterns here as you discover them during implementation._

- All touchable elements must have accessibilityLabel and accessibilityRole
- Use Zustand stores for state, not React context
- API responses always include processingMs for monitoring
- Moondream prompts live in `prompts/` directory, not inline
